# Ultrakill-Random-Level-Selector
This is a site that I made in like a week.
Maybe more I'm not sure.
It doesn't really have anything revolutinary or impressive that much but at least it get the work done.
So you can select any level/layer (except CyberGrind and Sandbox) in vanilla Ultrakill and then the site will choose a random one from the levels you choose.
